Problem Statement

System Architecture

Working Process

  • User submits a natural language query
  • Query is embedded using MiniLM
  • FAISS retrieves relevant context
  • LLM generates grounded text response
  • XTTS converts text to speech
  • SadTalker generates facial video output

Technologies Used

Python | MiniLM | FAISS | GPT4All | XTTS v2 | SadTalker | HTML | CSS | JavaScript
